Projects

Sports Party

Sports Party is a Mario Party-inspired sports-themed board game created using C++ and QT Creator. Players are greeted with a series of welcome dialogs that allow them to select the number of players (1-4) for their game session. Players are then able to roll a dice and traverse the game board, picking up power-ups, as they race to the finish. The game concludes when a player reaches the finish line. My partner and I created this as a class project.

Sports Party Repo

CSP Games

CSP games is a Java-based application with a Java Swing GUI that allows users to login in with a unique username and select one of three tabletop games to play against an NPC. The three games implemented are BlackJack, Tic-tac-toe, and Connect-4. Scores are then saved and stored in CSV-based leaderboard files. The application makes use of five object-oriented design patterns including MVC, strategy, composite, singleton, and observer. I developed this as a final project for an object-oriented class with a partner.

CSP Games Repo

2048 - Tris

This application provides a fun challenge for users. 2048 and Tetris are played by the user simultaneously using the same key inputs. The user's goal is to play for high scores against your friends and/or complete strangers. It is a full-stack web application developed using JavaScript, Node.JS, HTML, and CSS with a Firebase database to store user and leaderboard data. This was a group project completed alongside three other developers. My primary responsibility was designing the frontend pages, and connecting user login to the database.

DFS Inventory Tracker

DFS Inventory Tracker is a web app that can show a customer the inventory of the gas station they are at from the pump, eliminating the need to go inside only to find the item they were looking for out of stock or not sold at the gas station at all. Users can also play a matching game to earn loyalty points for the gas station. This project was developed as part of an internal hackathon hosted by the first company I interned for. My team created a Figma prototype and a basic interactive prototype using JavaScript.
